# TOG ASSESSMENT
### By: Muhammad Aria Muktadir

## Requirement
1. NodeJS
2. Git
3. npm/yarn

## How To Start Development

1. Clone this project
```bash
$ git clone https://github.com/muktadira22/tog-assessment/
```

2. Install Dependencies & Create Environment
```bash
$ cd tog-assessment
$ npm install
```

3. Run Development Server
```bash
$ npm run start
```
